############################
内容风格指南
############################

本项目使用rst作为内容的创作主要标记语言，另外可接受的语言是markdown。

****************
标题
****************

使用如下符号创建标题：

#. ``#`` with overline
#. ``*`` with overline
#. ``=``
#. ``-``
#. ``^``
#. ``"``

示例：

.. code-block:: rst

  ##################
  H1: document title
  ##################

  Introduction text.


  *********
  Sample H2
  *********

  Sample content.


  **********
  Another H2
  **********

  Sample H3
  =========

  Sample H4
  ---------

  Sample H5
  ^^^^^^^^^

  Sample H6
  """""""""

  And some text.


如果需要四级以上标题，考虑另起一个文档。
